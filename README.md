# 🦷 Dental Care 2.0



A modern and responsive dental clinic website designed to provide a professional online presence for dental practices. This updated version features an improved user interface, better user experience, mobile responsiveness, and optimized performance.



## ✨ Features



* Modern and clean design

* Fully responsive layout

* Appointment booking form

* Treatment & Services section

* Testimonials section

* Trust indicators

* Contact information

* Mobile-friendly interface

* Fast loading performance



## 🛠️ Technologies Used



* HTML5

* CSS3

* JavaScript
